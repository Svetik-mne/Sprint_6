# Sprint_6 — Автоматизация UI-тестов для сервиса «Яндекс.Самокат»

## Описание

Проект содержит автотесты для учебного сервиса [qa-scooter.praktikum-services.ru](https://qa-scooter.praktikum-services.ru), выполненные с использованием Selenium, Pytest и Allure. Реализован Page Object Model, тесты покрывают:

- Выпадающий список с вопросами/ответами
- Позитивный сценарий заказа самоката (2 набора данных, 2 кнопки заказа)
- Проверка переходов по логотипам «Самокат» и «Яндекс»

## Технологии

- Python 3.10+
- Selenium 4
- Pytest
- Allure
